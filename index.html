Pin Archiver

What is the Pin Archiver bot and what does it do?

Pin Archiver is a robust Discord pin bot. It automatically saves pinned messages into a designated “pin board”, or archive channel. The saved messages are sent in a neat embed and can be delivered through the bot or a webhook.

Setting up the bot

  1. Create an archive channel. Typically you want to disable users’ Send Messages permission.
  2. Optional: Create a webhook in this channel by going to Edit Channel > Integrations > Webhooks > New Webhook. Give it a nice name and profile picture.
  3. Use the .pin setarchive command to set the channel/webhook as the archive. Example: .pin setarchive #pins or .pin setarchive https://discordapp.com/api/webhooks/123456789/qwertyuiop123456789asdfghjklzxcvbnm ⚠️Warning: Please be cautious to not publicly share your webhook URL. If someone gets ahold of your webhook URL, they can send unwanted messages through the webhook. Run the command in a secret channel!⚠️
  4. Set a pretty color for the archive embeds. Example: .pin setcolor #be54d1. Use .pin color to send a sample embed to test out the color.
  5. Turn on automatic archiving for your desired channels. Example: Go into your #general and run .pin set on
  6. If you don’t want the archived messages to be kept in the actual Discord Pinned Messages, use .pin setunpin on. This is a server-wide setting.
  7. If you have pinned messages in a channel that you want to archive, run .pin archive and it will archive all current pinned messages.

Commands

Note: Commands below “.pin info” can only be performed by whitelisted users/roles or users with the Manage Server permission.

  • .pin help — List commands
  • .pin info — See info about this bot.
  • .pin setarchive [webhook|channel] — Set what webhook or channel to archive pins to. Archive channel will be set to none if no webhook or channel is given.
  • .pin set <on/off> — Enable/disable archiving from this channel
  • .pin status — Check if archiving is enabled for this channel
  • .pin setunpin <on/off> — Toggle whether or not the bot unpins a message after archiving it
  • .pin checkunpin — Check if automatic unpinning is enabled for this server
  • .pin archive [channel] — Archive all current pins in channel. The sent command’s channel’s pins will be archived if no channel is specified.
  • .pin force <message id> — Manually archive a message. This only works if sent in the same channel as the target message.
  • .pin setcolor <hex> — Set the sidebar color on Pin Archiver embeds in the server
  • .pin color — Returns the current color and sends a sample embed
  • .pin settings — See the server’s settings
  • .pin whitelist <add/remove> <role/user> — Manage the whitelist to execute restricted pin commands.
